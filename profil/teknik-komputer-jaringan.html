Teknik Komputer dan Jaringan (TKJ)

Lama Belajar: 3 Tahun

Teknik Komputer dan Jaringan (TKJ) adalah konsentrasi keahlian yang membekali peserta didik dengan kompetensi di bidang instalasi, konfigurasi, dan pemeliharaan perangkat keras komputer serta jaringan komputer. Melalui perpaduan pembelajaran teori dan praktik langsung di laboratorium, siswa dipersiapkan untuk mampu merancang, membangun, mengelola, dan mengamankan jaringan komputer sesuai dengan standar kompetensi kerja yang berlaku di dunia usaha dan dunia industri (DUDI).

Konsentrasi/Peminatan yang Dipelajari

  1. Instalasi dan konfigurasi jaringan komputer (LAN, WAN, dan Wireless Network)
  2. Administrasi sistem operasi jaringan dan server
  3. Keamanan jaringan (network security) dan troubleshooting jaringan

Bahasa/Teknologi yang Dipelajari

  1. Konfigurasi perangkat Mikrotik RouterOS dan Cisco (Packet Tracer)
  2. Administrasi server berbasis Linux (Debian/Ubuntu Server) dan Windows Server
  3. Konsep TCP/IP, subnetting, dan routing (statis maupun dinamis)
  4. Virtualisasi jaringan menggunakan Proxmox/VirtualBox/GNS3

Prospek Kerja di Dunia Industri

  1. Network Administrator / Network Engineer
  2. Teknisi Jaringan Komputer
  3. IT Support / IT Helpdesk
  4. Wirausaha di bidang jasa instalasi dan perawatan jaringan komputer

Sarana yang Dimiliki

  1. Laboratorium komputer dan jaringan
  2. Perangkat jaringan praktik (router, switch, access point)
  3. Server praktik untuk simulasi administrasi jaringan
  4. Alat instalasi jaringan (crimping tool, cable tester, tang crimping)
